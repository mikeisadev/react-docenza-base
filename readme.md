# Repo GitHub sulle docenze base di React

In questa repository è possibile trovare la raccolta di vari progetti e spiegazioni base su ReactJS

## Come navigare questa repository

Per navigare questa repository devi basarti su tutte le cartelle che vedi.

Ogni cartella rappresenta un progetto React indipendente.

Puoi tranquillamente scaricare questa repository oppure studiare e vedere gli esempi di codice direttamente da GitHub.
